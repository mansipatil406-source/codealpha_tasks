## 📌 Description

This project is a simple Banking System developed in C using file handling. It allows users to manage bank accounts by performing operations such as creating an account, depositing money, withdrawing money, and checking account details.

---

## ✨ Features

* 🆕 Create Account
* 💰 Deposit Money
* 💸 Withdraw Money
* 🔍 Balance Enquiry
* 📋 Display All Accounts
* 💾 File Handling for data storage

---

## ⚙️ Technologies Used

* C Programming
* File Handling
* Structures
