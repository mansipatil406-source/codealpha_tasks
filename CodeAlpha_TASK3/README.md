## 📌 Description

This project is a Student Management System developed in C using file handling. It allows users to store and manage student records such as ID, name, and marks.

---

## ✨ Features

* ➕ Add Student
* 📋 Display All Students
* 🔍 Search Student by ID
* ✏️ Update Student Record
* ❌ Delete Student Record
* 💾 File Handling for data storage

---

## ⚙️ Technologies Used

* C Programming
* File Handling
* Structures
