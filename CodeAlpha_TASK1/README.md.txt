# Task 1: Simple Calculator in C

## 👩‍💻 Author:

Mansi Patil

---

## 📌 Description:

This program is a simple calculator developed in C that performs basic arithmetic operations using a menu-driven approach. The user selects an operation and inputs two numbers to get the result.

---

## ✨ Features:

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero error
* User-friendly menu system

---

## ⚙️ Technologies Used:

* C Programming

---

## ▶️ How to Run:

1. Open terminal / command prompt

2. Compile the program:
   gcc Task1_Calculator.c

3. Run the program:
   ./a.out

---

## 📂 File Included:

* Task1_Calculator.c

---

## 📸 Output:

Displays result based on selected operation.

---

## ✅ Conclusion:

This project demonstrates the use of conditional statements, switch-case, and basic input/output operations in C.
